Detailed Description

template<typename Key, typename Value, typename Ttl>
class IKVStorage< Key, Value, Ttl >

Интерфейс хранилища ключ-значение.

Определяет базовые операции для работы с KV-хранилищем: добавление, удаление, получение значений, получение нескольких записей в отсортированном виде и удаление протухших записей.

Template Parameters
KeyТип ключей.
ValueТип значений.
TtlТип для хранения времени жизни записи.

Получает несколько записей, начиная с указанного ключа, в лексикографическом порядке.

Parameters
keyКлюч, с которого начинать поиск (включительно).
countМаксимальное количество записей для получения.
Returns
Вектор пар ключ-значение в лексикографическом порядке.

Пример: ("a", "val1"), ("b", "val2"), ("d", "val3"), ("e", "val4") getManySorted("c", 2) -> ("d", "val3"), ("e", "val4")
