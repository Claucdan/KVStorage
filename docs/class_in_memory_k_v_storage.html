Detailed Description

template<typename Key, typename Value, typename Ttl, typename Clock>
class InMemoryKVStorage< Key, Value, Ttl, Clock >

Хранилище ключ-значение в памяти.

Предоставляет хранение пар ключ-значение с поддержкой опционального TTL (время жизни записи). Поддерживаются базовые операции: добавление, получение, удаление и получение нескольких записей в отсортированном виде.

Template Parameters
KeyТип ключей.
ValueТип значений.
TtlТип для хранения времени жизни записи.
ClockТип часов, используемых для определения текущего времени.
